# Spotify Ads Agent Corpus

A source-backed, agent-oriented collection of public Spotify Advertising Canada material about ad formats, specifications, creative practice, campaign operation, targeting, measurement, programmatic buying, industry uses, case studies, and audience trends.

## What is included

- 151 separate transformed Markdown records from live web pages.
- 8 separate transformed Markdown records for first-party linked PDFs.
- 159 source records total.
- A consolidated playbook, agent system prompt, inventory, crawl-gap report, JSONL manifest, and SHA-256 checksums.

## Content mode

This is not a verbatim site mirror. Each record preserves provenance, headings, retrieval cues, structured technical markers, and case-study fields while paraphrasing or abstracting long source prose. The live source URL remains attached to every record.

## Recommended retrieval order

1. Use `web/ad-specs/` and `web/help-center/` for current operational constraints.
2. Use `web/best-practices/` for creative recommendations.
3. Use `web/goals-measurement-targeting/` and `web/programmatic/` for campaign design and buying paths.
4. Use `web/case-studies/` for examples, never as universal causal proof.
5. Use `web/trends-research/` and `documents/` for dated research signals with market/sample caveats.

## Completeness boundary

The corpus covers the working, query-free Canadian URL graph found through current navigation, recursive internal links, and multiple search-index cross-checks on 2026-07-21. The localized sitemap repeatedly returned a 504, so this is a comprehensive evidence-backed collection, not proof of a complete historical mirror. See `COVERAGE_GAPS.md`.
